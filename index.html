Linux crontab命令

发表于 2016-11-22 分类于 Linux

Linux crontab命令

crontab命令选项:

     -u指定一个用户
     -l列出某个用户的任务计划
     -r删除某个用户的任务
     -e编辑某个用户的任务

cron文件语法:

      分     小时    日       月       星期     命令
      0-59   0-23   1-31   1-12     0-6     command     (取值范围,0表示周日一般一行对应一个任务)
     记住几个特殊符号的含义:
         “*”代表取值范围内的数字,
         “/”代表”每”,
         “-”代表从某个数字到某个数字,
         “,”分开几个离散的数字

y=`date "+%Y"`
m=`date "+%m"`
d=`date "+%d"`

cp catalina.out catalina.out.$y$m$d.log

exit

  4. 接下来我们选择“滤镜——其他——高反差保留”，点击“确定”
  5. 选择“图像——应用图像——叠加——确定”，这里可以重复两次的。
  6. 选择“图像——应用图像——颜色减淡——确定“
